
# Trains-table is a test React application.

React app demonstration.


## Installation

Clone my project, after type in terminal:

```bash
  npm install 
  npm run dev
```

## Tech Stack

HTML, CSS, React, Redux, TS, Bootstrap-icons, Axios, datepicker, date-fns.
